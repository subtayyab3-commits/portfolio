# Tayyab AI Portfolio

A  animated business-style portfolio website built with HTML, CSS, and JavaScript.  
Includes interactive sections for freelance services, AI tools, and automation demos.

## Features
- Responsive 3D design
- Animated sections using Kiwarp / Three.js
- Contact form integration
- English default language

## Run Locally
1. Download the project files
2. Open `index.html` in any browser
